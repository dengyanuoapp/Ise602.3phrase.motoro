Verilog - Cadence Xcelium

Verilog is a hardware description language (HDL) for developing and modeling circuits. The Cadence Xcelium tool will help you simulate circuits that have been developed in Verilog.

To use the tool, start up your X-Windows emulator to get an X-terminal window. At the X-terminal window, you may create Verilog files and run the Verilog program as directed in the tutorial below.

Tutorial

Use the file half_adder.v as an example file for this tutorial.

To run the Verilog program using this file, use the command: xmverilog half_adder.v

The program will print the results on the console. The results will also be written to the xmverilog.log file.

To exit the program, use the command: exit
